TypeScript

是什么？

TypeSctipt 是微软公司开发的一款开源的JavaScript 超集语言
JavaScript超集：当前任何JavaScript 都是合法的TypeScript代码
TypeScript 主要为JavaScript 提供了类型系统和ES6语法的支持

flow 是一个类型检查工具， TypeScript 是一门开发语言
TypeScript 有自己的编译工具，我们写好的TypeScript 代码最终会通过编译器编译为JavaScript代码运行
