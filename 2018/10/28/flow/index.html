###总结

##flow 是什么
Static Type Checker For JavaScript
静态类型检查工具

##flow 能做什么
Flow 能够给JavaScript 提供静态类型检查的能力，其实就是为JavaScript添加了一个编译过程

##Flow 的使用

1.安装flow
npm i flow-bin -D
2.需要编写flow代码
通过注释的方式为代码添加类型（不会对js代码产生任何更改，影响） 通过直接在js代码中书写类型（改变了js代码的结构，需要通过babel进行转码之后，才能正常运行）
`js
// 文件中需要添加一个标记 告诉flow当前文件需要检查
// @flow
