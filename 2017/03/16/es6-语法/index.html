Es6语法

var、let、const 区别

在 es5中 var 可以用来声明一个变量 如果在{}里声明的 全局都可以访问 容易造成内存泄漏

在es6中 let const 和 block 作用域

{} 可以形成一个块级作用域

let 允许 创建块级作用域，ES6推荐在函数中使用let 定义变量，而不是var

同样在块级作用域中有效的是const 它是专用于声明一个常量的，在es6中声明常量类似于一个指针，它指向某个地址（引用）也就是说常量并非是不能被改变的

有几个注意点

  • let 关键词声明的变量不具备变量提升的特性
  • let和const声明只在最靠近的一个块中（花括号内）有效
  • 当使用常量const 声明时，请使用大写变量 如DB
  • const 在声明时必须被赋值

##箭头函数

在es6中，箭头函数则是函数的一种简写形式，使用括号包裹参数，跟随一个=> 紧跟这是函数体

当然，箭头函数不仅仅是让代码变得简介，函数中this总是绑定指向对象自身

一般情况下我们会用一个变量来保存this 然后在getUp中使用

然而箭头函数可以省去这个

模版语法和分隔符

ES6中 有一种非常简洁的方法组装一堆字符串和变量

  • ${…} 用来渲染一个变量

  • `作为分隔符

##for…of 和for…in

for..of 用于遍历一个迭代器 如数组

for… in 用来遍历对象中的属性
